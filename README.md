# DAF-EfficientNetV2: A Novel Approach for Plant Leaf Disease Recognition via Distribution-Aware Feature Enhancement on EfficientNetV2

This repository provides the framework and partial implementation of the method proposed in "DAF-EfficientNetV2: A Novel
Approach for Plant Leaf Disease Recognition via Distribution-Aware Feature Enhancement on EfficientNetV2", which is
currently under editorial review. The complete implementation will be released upon acceptance.

Example dataset placement:

- `data/`
  - `train/`
    - `apple_scab/`
    - `Black_rot/`
    - `cedar_apple_rust/`
    - `healthy/`
  - `val/`
    - `apple_scab/`
    - `Black_rot/`
    - `cedar_apple_rust/`
    - `healthy/`

Our final training weights will be made public after the paper is accepted
